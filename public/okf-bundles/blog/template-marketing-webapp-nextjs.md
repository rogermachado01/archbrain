This concept represents a marketing web application built with Next.js, serving as a frontend application within an architecture diagram. It denotes the presence of a customer-facing or promotional website layer built on the Next.js framework, distinct from other application types such as backend services or internal tools.

As a Frontend Application type, this concept is used to model the marketing-focused web presence of a system in architecture diagrams, helping teams visualize where the public-facing Next.js site fits relative to other components in the overall system design.
